Accessible Fill-in-the-Blank Activity Creator

Teachers can create multi-sentence activities, save them in the browser, and export a standalone student file.

How to create an activity

Enter each sentence with [blank] where the missing word should go. Provide several word options, and choose the correct answer.

The exported student file will present one sentence at a time for students to complete.

Add a new sentence

Example: The cat chased the [blank] across the yard.
Enter options separated by commas. Include the correct answer in the list if applicable.
